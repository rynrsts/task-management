# Task Management

Web backend application for managing tasks with users

<br>

## Project setup

```bash
$ yarn install
```

### Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

<br>

## Languages and Tools

- VSCode
- NestJs
- TypeORM
- pgAdmin
